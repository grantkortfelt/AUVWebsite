About the Software Team

The Software team focuses on developing the algorithms and software that control the AUV. We work on navigation, obstacle avoidance, and task execution.

Our team is dedicated to creating efficient and reliable software that enables the AUV to operate autonomously in complex underwater environments.

Key Responsibilities

The Software team is responsible for the following tasks:
* Developing the AUV's control algorithms
* Implementing navigation and mapping software
* Ensuring software reliability and performance
* Collaborating with other teams to integrate hardware and software components

Team Goals

The Software Team aims to develop innovative solutions for autonomous underwater vehicles, focusing on efficiency, reliability, and cutting-edge technology.
